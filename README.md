# Analyse de Fourier & Traitement du Signal Audio

Ce projet explore les fondements de la **Transformée de Fourier Discrète (TFD)** et son application pratique au traitement de signaux sonores sous Python.

## Fonctionnalités
- **Analyse Fréquentielle** : Décomposition de signaux complexes et visualisation de spectres d'amplitude.
- **Manipulation de Notes** : Algorithme de décalage fréquentiel pour modifier la hauteur d'une note sans changer sa durée.
- **Effets Audio** : Implémentation d'effets d'**écho** et de **réverbération** en utilisant la convolution fréquentielle (FFT).



## Installation & Usage
1. Cloner le dépôt.
2. Installer les dépendances : `pip install numpy matplotlib scipy sounddevice soundfile`.
3. Lancer `python Ma312_BE_main.py` pour tester les effets sur un fichier `.wav`.
